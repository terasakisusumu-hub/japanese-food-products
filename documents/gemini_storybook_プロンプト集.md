# 自転車練習ストーリーブック用プロンプト集

Geminiにそのままコピー＆ペーストして使えるプロンプトをまとめました。

---

## 📘 プロンプト1：基本のストーリー生成

```
あなたは子ども向け絵本作家です。
小学2年生の男の子が自転車の練習を楽しく続けられるような、やさしい絵本のストーリーを作ってください。

【主人公】
- タコのぬいぐるみ「タコくん」：勇気があって前向き、でも最初は自転車が怖い
- レッサーパンダのぬいぐるみ「パンちゃん」：優しくて、いつも友だちを励ましてくれる

【大切な設定】
- タコくんは自転車のペダルに足を乗せられるのは、まだほんの一瞬（1〜2秒）
- 何回も転んで、膝を擦りむいたり、手に傷ができたりする
- パンちゃんがそばで励まし続ける
- 少しずつ上達していく過程を丁寧に描く
- 最後には風を切って自転車に乗れるようになる

【形式の指定】
- 8ページ構成
- 各ページに「シーンの説明」と「セリフ」を入れる
- ひらがなを多めに使い、小学2年生が読みやすい文章にする
- 「転んでも大丈夫」「少しずつできるようになる」というメッセージを込める

絵本を書いてください。
```

## 📘 プロンプト2：詳細なシーン指定版

```
以下の8つのシーンで、自転車練習の絵本ストーリーを作ってください。

【登場人物】
・タコくん：主人公。自転車に乗れるようになりたい。
・パンちゃん：タコくんの親友。いつも応援してくれる。

---

【ページ1：はじまり】
タコくんは、お友だちが自転車でスイスイ走るのを見て、うらやましく思っています。
「ぼくも、あんなふうに走りたいなあ...」
パンちゃんが言いました。「いっしょに れんしゅう しよう！」

【ページ2：さいしょの いっぽ】
タコくんが自転車にまたがります。
ペダルに足を乗せて...「1...2...」
グラグラグラ...ドテン！
転んでしまいました。

【ページ3：いたいけど】
「いたたた...」ひざに すりきずが できました。
タコくんは泣きそうです。
でもパンちゃんが言いました。
「だいじょうぶ！ころんだのは、がんばった しょうこ だよ！」

【ページ4：もういちど】
タコくんは涙をふいて、また自転車に乗ります。
「1...2...3...」今度は3秒のれました！
「やった！さっきより ながい！」

【ページ5：まいにち すこしずつ】
毎日、練習しました。
3秒が5秒に、5秒が10秒になりました。
ばんそうこうも増えたけど、えがおも増えました。
「きょうは きのうより ながく のれた！」

【ページ6：あきらめない】
何度も転びました。何度も立ち上がりました。
パンちゃんは いつも そばにいてくれました。
「タコくん、すごいよ。ぜったい のれるようになる！」

【ページ7：のれた！】
ある日のこと。
タコくんがペダルをこぎます。1かい、2かい、3かい...
自転車が走りだしました！
「のれた！のれた！ぼく、じてんしゃ のれてる！」

【ページ8：おわり】
風がきもちいい。
タコくんは、自転車が だいすきに なりました。
「ころんだ きずあとは、がんばった くんしょう だね！」
パンちゃんと タコくんは、笑いました。

おしまい。

---

この流れで、セリフと描写を膨らませて、絵本らしい文章にしてください。
各ページは80〜120文字程度でまとめてください。
```

## 📘 プロンプト3：お子さんの名前入りバージョン

```
小学2年生の男の子「〇〇くん」（←お子さんの名前に変えてください）が読む、自転車練習の絵本を作ってください。

【登場人物】
・タコくん：勇気があるタコのぬいぐるみ
・パンちゃん：やさしいレッサーパンダのぬいぐるみ
・〇〇くん：タコくんとパンちゃんの友だち（読んでいる子ども自身）

【ストーリーの流れ】
1. 〇〇くんが自転車に乗りたいと思っている
2. タコくんとパンちゃんが「一緒に練習しよう！」と声をかける
3. みんなで練習を始める
4. 最初はペダルに足を乗せるのも一瞬だけ
5. 何度も転ぶけど、仲間が励ましてくれる
6. 少しずつ上達していく
7. ついに自転車に乗れるようになる！
8. 「次はどこまで行こうか？」とみんなでわくわくする

【大切にしてほしいこと】
・「〇〇くん」が物語の中で励まされ、自分も頑張ろうと思える
・失敗は成長の一部だというメッセージ
・小学2年生が読める、やさしい日本語で

絵本を書いてください。
```

## 📝 追加カスタマイズ用プロンプト

### 励ましの言葉を追加したい場合

```
上で作ったストーリーに、以下の励ましの言葉を自然に入れてください：

1.「できないことが できるようになるのは、すごいこと！」
2.「ころぶのは、チャレンジしている しょうこ」
3.「きのうより きょう、きょうより あした」
4.「いたくても、なきたくても、あきらめなかった きみは つよい」
```

### 実際の練習場所を入れたい場合

```
このストーリーの練習場所を「〇〇公園」（←実際の公園名）に変更してください。
公園の特徴（例：大きな木がある、噴水がある、など）も描写に入れてください。
```

### お父さん・お母さんを登場させたい場合

```
ストーリーに「おとうさん」か「おかあさん」を追加してください。
タコくんが転んだときに優しく声をかけたり、
自転車の後ろを支えて一緒に練習してくれるシーンを入れてください。
```

## 💡 うまくいくコツ

1. **最初は基本プロンプトから試す** → 出てきた結果を見て調整
2. **「もっと〇〇して」と追加指示** → 長くしたい、短くしたい、励ましを増やしたい など
3. **お子さんと一緒に作る** → 「タコくんに何て言ってほしい？」と聞きながら
