# NotebookLM用プロンプト集
## 蛇口規格（W26山20、G1/2、水栓呼び13）の資料作成

### 基本プロンプト

#### 1. 概要の生成
```
この動画の内容を基に、蛇口のネジ規格（W26山20、G1/2、水栓呼び13）について、初心者にもわかりやすく説明してください。
```

#### 2. 比較表の作成
```
動画で説明されているW26山20とG1/2の違いを、以下の項目で比較表を作成してください：
- 外径
- ネジ山数
- ネジピッチ
- 主な用途
- 互換性
- 視覚的な見分け方
```

#### 3. 水栓呼びとの関係
```
水栓呼び13とは何か、そしてW26山20やG1/2とどのような関係があるのか、動画の内容を基に詳しく説明してください。
```

### 詳細プロンプト

#### 4. 実用的な情報の抽出
```
この動画から、実際のDIY作業や部品交換で役立つ以下の情報を抽出してください：
- 間違った接続を避ける方法
- 規格を見分けるコツ
- 必要な工具
- よくある間違いとその対処法
```

#### 5. 視覚的な説明の強化
```
動画で説明されている蛇口パーツの形状やサイズについて、以下のような比喩や具体例を使って説明してください：
- コインとの比較（1円玉、500円玉など）
- 日常的な物との比較
- 実際の使用シーン
```

#### 6. タイムスタンプ付きセクション
```
動画の重要なポイントを、タイムスタンプ付きで箇条書きにしてください。
特に以下のトピックについて：
- W26山20の説明
- G1/2の説明
- 水栓呼び13の説明
- 実際の接続方法
- 注意点やトラブルシューティング
```

### 応用プロンプト

#### 7. FAQ形式での整理
```
動画の内容を基に、蛇口規格に関するよくある質問（FAQ）を5〜10個作成し、それぞれに回答してください。
```

#### 8. ステップバイステップガイド
```
動画で説明されている蛇口パーツの交換や接続方法を、初心者向けのステップバイステップガイドとして整理してください。
各ステップに注意点も含めてください。
```

#### 9. トラブルシューティング
```
動画で説明されている、または動画から推測できる、蛇口規格に関するトラブルとその解決方法をまとめてください。
```

#### 10. 用語集の作成
```
動画で使われている専門用語（W26山20、G1/2、水栓呼び13、PF1/2、管用平行ネジなど）を、わかりやすく説明した用語集を作成してください。
```

### 統合プロンプト

#### 11. 既存資料との統合
```
この動画の内容と、以下の情報を統合して、包括的な蛇口規格ガイドを作成してください：

[ここに既存のfaucet_specs_comparison.mdの内容を貼り付け]

動画の視覚的な説明と、テキスト資料の詳細な数値情報を組み合わせて、わかりやすい資料にしてください。
```

#### 12. 図解の説明
```
動画で示されている蛇口パーツの形状や接続方法について、文章で詳しく説明してください。
読者が動画を見なくても理解できるように、視覚的な情報を言葉で補足してください。
```

### 特定の用途向けプロンプト

#### 13. DIY初心者向け
```
この動画の内容を、DIY初心者でも理解できるように、専門用語を避け、わかりやすい言葉で説明してください。
特に、なぜ規格が違うのか、なぜ互換性がないのかを、理由も含めて説明してください。
```

#### 14. プロ向け（詳細仕様）
```
動画の内容を基に、プロの水道工事士やDIY上級者向けに、以下の詳細情報を整理してください：
- JIS規格番号
- 正確な寸法（公差を含む）
- 材質の推奨事項
- 締め付けトルク
- シール材の種類
```

#### 15. 教育用資料
```
この動画の内容を、学校や職業訓練で使える教育用資料として整理してください。
学習目標、重要ポイント、確認問題を含めてください。
```

### プロンプトの使い方のコツ

1. **段階的に質問する**
   - まず概要を取得
   - 次に詳細を深掘り
   - 最後に統合

2. **具体的な形式を指定する**
   - 「表形式で」「箇条書きで」「図解付きで」など

3. **対象読者を明確にする**
   - 「初心者向けに」「プロ向けに」など

4. **既存情報と組み合わせる**
   - テキスト資料や画像も一緒に追加して統合

### 動画がない場合の代替プロンプト

#### テキストソース用
```
以下の情報を基に、W26山20、G1/2、水栓呼び13の関係について、わかりやすい説明資料を作成してください。

[既存のテキスト資料を貼り付け]
```

#### 画像ソース用
```
追加した画像を基に、蛇口パーツの規格の違いを説明してください。
特に、サイズの違い、ネジの形状の違い、使用場所の違いについて詳しく説明してください。
```

---

**使用方法：**
1. NotebookLMに動画やテキストを追加
2. 上記のプロンプトをコピー＆ペースト
3. 必要に応じてカスタマイズ
4. 生成された内容を確認・編集
5. エクスポートして最終資料を作成
