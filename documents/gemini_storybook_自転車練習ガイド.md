# Gemini ストーリーブック機能で作る「自転車練習ストーリーブック」ガイド

## 🎯 このガイドについて

小学2年生のお子さんが自転車の練習を楽しく続けられるよう、Geminiのストーリーブック機能を使って、お気に入りのぬいぐるみ（タコとレッサーパンダ）が主人公の物語を作る方法を解説します。

---

## 📖 Geminiのストーリーブック機能とは

Googleの **Gemini** には、画像とテキストを組み合わせてストーリーを生成する機能があります。特に **Google の ImageFX** や **Whisk** といったツールと連携することで、アップロードした写真をもとにキャラクターを作成し、絵本のような物語を生成できます。

---

## 🚀 利用手順

### ステップ1：Geminiにアクセスする

1. ブラウザで **[gemini.google.com](https://gemini.google.com)** にアクセス
2. Googleアカウントでログイン
3. 画像生成やストーリー作成には **Gemini Advanced**（有料プラン）がおすすめですが、無料版でもテキストベースのストーリーは作成可能

### ステップ2：ぬいぐるみの画像を準備する

**撮影のコツ：**
- 明るい場所で、ぬいぐるみがはっきり写るように撮影
- できれば背景はシンプルに（白や単色の壁など）
- タコとレッサーパンダ、それぞれ単体の写真を用意
- 表情が見えるアングルで撮影

### ステップ3：Geminiでストーリーを作成する

#### 方法A：Geminiに直接依頼する

Geminiのチャット画面で、以下のようなプロンプトを入力します：

### 💬 基本プロンプト（コピーして使えます）

```
小学2年生の男の子向けに、自転車の練習を題材にした絵本のストーリーを作ってください。

【登場キャラクター】
- タコのぬいぐるみ「タコくん」：勇気があって、何度転んでも立ち上がる
- レッサーパンダのぬいぐるみ「パンちゃん」：優しくて、いつも応援してくれる

【ストーリーの要素】
1. 最初は自転車のペダルに足を乗せられるのがほんの一瞬
2. 何度も転んで、膝を擦りむいたりする場面がある
3. でも諦めずに練習を続ける
4. 友達が応援してくれる
5. 最後には自転車に乗れるようになって大喜び

【形式】
- 8〜10ページ程度の絵本形式
- 各ページには場面の説明と、キャラクターのセリフを入れる
- 小学2年生が読める、ひらがな多めのやさしい言葉で
- 励ましや頑張りを認める言葉を入れる
```

## 🎨 画像付きストーリーブックを作る方法

### 方法B：Whisk（Google Labs）を使う

**Whiskとは：** Googleが提供する実験的な画像生成ツールで、自分の写真をアップロードしてキャラクター化できます。

1. **[labs.google/whisk](https://labs.google/whisk)** にアクセス
2. 「Subject」にぬいぐるみの写真をアップロード
3. 「Scene」で背景を指定（例：公園、道路など）
4. 「Style」で絵本風のスタイルを選択
5. プロンプトで具体的なシーンを指定

**Whiskでの各シーン生成プロンプト例：**

```
【シーン1用】
Subject: タコのぬいぐるみの写真
Scene: 公園の芝生、晴れた日
Style: 子供向け絵本イラスト風
Prompt: A cute octopus plushie character sitting on a small bicycle in a park, looking nervous but excited, children's book illustration style

【シーン2用】
Subject: タコのぬいぐるみの写真
Scene: 公園の道
Style: 子供向け絵本イラスト風
Prompt: A cute octopus plushie character falling off a bicycle, surprised expression, band-aid on knee, children's book illustration style

【シーン3用】
Subject: レッサーパンダのぬいぐるみの写真
Scene: 公園のベンチ
Style: 子供向け絵本イラスト風
Prompt: A cute red panda plushie character encouraging a sad octopus plushie, warm and supportive scene, children's book illustration style
```

## 📝 完成したストーリーの活用方法

### 1. 印刷して絵本にする

- Geminiで生成した文章と、Whiskで生成した画像を組み合わせる
- Canva や PowerPoint で簡単に絵本レイアウトを作成
- A4用紙に印刷して、製本テープで綴じれば手作り絵本に！

### 2. デジタル絵本として楽しむ

- Google スライドやPowerPointでスライドショー形式に
- タブレットやPCで読み聞かせ

### 3. 読み聞かせ音声をつける

- NotebookLM を使って音声化
- お子さんの声で録音して、よりパーソナルに

## 🌟 ストーリーをさらに良くするヒント

### お子さんの名前を入れる

```
登場キャラクターに「○○くん（お子さんの名前）」という人間の友達を追加してください。
タコくんとパンちゃんが○○くんと一緒に自転車の練習をする物語にしてください。
```

### 実際のエピソードを反映する

```
【追加してほしいシーン】
- 練習中にアイスを食べて休憩する場面
- お父さんやお母さんが後ろを支えてくれる場面
- 〇〇公園で練習している設定にする
```

### 励ましのメッセージを強調する

```
ストーリーの中に、以下のメッセージを自然に入れてください：
- 「できないことが、できるようになるのは、すごいこと」
- 「転ぶのは、チャレンジしている証拠」
- 「少しずつ上手になるって、とっても楽しい」
```
