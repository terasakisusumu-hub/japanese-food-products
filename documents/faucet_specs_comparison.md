# 蛇口・配管ネジ規格比較ガイド：G1/2 vs W26山20

このドキュメントは、水回りのDIYや部品交換で間違いやすいネジ規格、特に「G1/2」と「W26山20」の違いについて詳細にまとめたものです。

## 1. 概要・主な用途

| 規格名 | 通称・別名 | 主な用途 |
| :--- | :--- | :--- |
| **G1/2** | PF1/2, 管用平行ネジ(呼び13) | シャワーホースの接続、トイレの給水管、フレキ管の接続など |
| **W26山20** | W26-20, 26ミリネジ | 混合水栓のスパウト（吐水パイプ）の根元接続（日本独自規格に近い） |

> **注意**: どちらも「呼び13（13mm用）」の水栓器具に関連しますが、**接続する場所**が全く異なります。

## 2. 形状・寸法データ

| 項目 | G1/2 (PF1/2) | W26山20 |
| :--- | :--- | :--- |
| **ネジ外径** | 約 20.9 mm (約21mm) | 26.0 mm |
| **ネジ山数** | 1インチあたり 14山 | 1インチあたり 20山 |
| **ネジピッチ** | 約 1.814 mm | 1.270 mm |
| **規格の由来** | JIS B 0202 (ISO 228-1) 国際的 | JIS規格外（日本水栓独自の慣用規格） |
| **特徴** | ネジの山が少し粗い | ネジの山が細かい（密になっている） |

## 3. 視覚的な見分け方

1.  **太さの違い**:
    *   **W26山20**の方が明らかに太いです（直径26mm）。500円玉（26.5mm）とほぼ同じ大きさです。
    *   **G1/2**は一回り細いです（約21mm）。1円玉（20mm）よりわずかに大きいくらいです。

2.  **場所による判断**:
    *   **壁から出ているクランクと水栓本体の間**: 多くは **G3/4** (大きい)
    *   **水栓本体とシャワーホースの間**: 多くは **G1/2**
    *   **水栓本体と吐水パイプ（スパウト）の間**: 国産の一般的な混合栓なら **W26山20**

## 4. 変換が必要なケース

*   **海外製シャワーヘッドへの交換**: 海外製はG1/2が標準ですが、日本の古いメーカー（KVK, MYMなど）は独自の細目ネジ（M22x2など）を使っている場合があり、アダプターが必要です。
*   **パイプの延長**: 水栓のパイプを上向きにしたり延長したりする場合、根元のネジ規格（W26山20）を確認する必要があります。

## 5. NotebookLLMなどでの活用用サマリー

*   **G1/2**: 「管用平行ネジ」。水圧がかかる配管接続用。世界共通規格。ネジ山は粗い。
*   **W26山20**: 「水栓スパウト用特殊ネジ」。日本の水栓金具のパイプ接続用。ネジ山は細かい。
*   **互換性**: **全くありません**。無理にねじ込むとネジ山を潰して水漏れの原因になります。
