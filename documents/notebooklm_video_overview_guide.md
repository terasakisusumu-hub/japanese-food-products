# NotebookLM Video Overview機能の使い方ガイド
## 蛇口規格（W26山20、G1/2、水栓呼び13）の資料作成

### NotebookLMとは
NotebookLMは、Googleが提供するAIを活用した研究・学習支援ツールです。動画や音声、テキストなどのソースから自動的に要約や概要を生成できます。

### Video Overview機能の基本的な使い方

#### 1. NotebookLMへのアクセス
- **URL**: https://notebooklm.google.com/
- Googleアカウントでログインが必要です

#### 2. 新しいノートブックの作成
1. NotebookLMのホーム画面で「**+ New notebook**」をクリック
2. ノートブックに名前を付けます（例：「蛇口規格ガイド」）

#### 3. 動画ソースの追加
1. ノートブック内の「**Add source**」または「**+**」ボタンをクリック
2. ソースタイプから「**Video**」を選択
3. 以下のいずれかの方法で動画を追加：
   - **YouTube URL**: 蛇口の規格説明動画のURLを貼り付け
   - **動画ファイルのアップロード**: ローカルの動画ファイルをアップロード
   - **Google Drive**: Google Driveに保存されている動画を選択

#### 4. Video Overviewの生成
動画を追加すると、NotebookLMが自動的に：
- **動画の要約**を生成
- **主要なポイント**を抽出
- **タイムスタンプ付きのセクション**を作成

#### 5. 蛇口規格に関する動画の推奨内容
以下のような内容の動画を用意すると効果的です：

**推奨動画内容：**
- 実際の蛇口パーツ（W26山20、G1/2）を並べて比較
- ネジの形状を拡大して見せる
- 水栓呼び13の意味を説明
- 実際の接続方法をデモンストレーション
- 間違った接続によるトラブルの例

**動画の長さ：**
- 5〜15分程度が最適（長すぎると処理に時間がかかります）

### 蛇口規格資料作成の具体的な手順

#### ステップ1: 動画の準備
1. **実物を使った説明動画を作成**
   - W26山20とG1/2のパーツを並べて撮影
   - 定規やコイン（1円玉、500円玉）を使ってサイズ比較
   - ネジの山の細かさを拡大して見せる

2. **または既存のYouTube動画を探す**
   - 「蛇口 ネジ 規格」「水栓 呼び13」「G1/2 W26」などのキーワードで検索
   - DIYチャンネルや水道工事の解説動画を探す

#### ステップ2: NotebookLMでの処理
1. 動画をNotebookLMに追加
2. 自動生成される概要を確認
3. 必要に応じて以下のプロンプトで追加情報を取得：
   - 「W26山20とG1/2の違いを表形式でまとめて」
   - 「水栓呼び13とは何か、わかりやすく説明して」
   - 「実際の使用例を箇条書きで教えて」

#### ステップ3: 生成された資料の活用
1. **概要セクション**を確認
   - 動画の主要ポイントが自動抽出されています
2. **タイムスタンプ**を活用
   - 重要な部分への直接リンクが生成されます
3. **要約をエクスポート**
   - 生成された内容をコピーして、MarkdownやHTMLに整理

#### ステップ4: 既存資料との統合
既存の`faucet_specs_comparison.md`と組み合わせて：
- 動画の視覚的な説明
- テキスト資料の詳細な数値情報
- 実物写真や図解

これらを統合した包括的な資料を作成できます。

### プロンプト例（NotebookLMで使用）

動画を追加した後、以下のようなプロンプトを試してみてください：

```
この動画の内容を基に、W26山20とG1/2の違いを初心者にもわかりやすく説明してください。
特に、水栓呼び13との関係についても触れてください。
```

```
動画で説明されている蛇口の規格について、表形式で比較表を作成してください。
寸法、用途、互換性について整理してください。
```

```
この動画の内容を基に、実際のDIY作業で注意すべきポイントを箇条書きでまとめてください。
```

### 注意点

1. **動画の品質**
   - 音声がはっきり聞こえること
   - 視覚的にわかりやすいこと
   - 日本語の説明があること（または字幕があること）

2. **処理時間**
   - 動画の長さによって処理に数分かかる場合があります
   - 長い動画（30分以上）は分割することを推奨

3. **プライバシー**
   - 動画をアップロードする場合、公開設定に注意
   - 個人情報が含まれていないか確認

### 代替案：動画がない場合

動画がない場合は、以下の方法も有効です：

1. **テキストソースとして既存資料を追加**
   - `faucet_specs_comparison.md`をNotebookLMに追加
   - 「わかりやすい説明資料を作成して」とプロンプト

2. **画像を追加**
   - 蛇口パーツの写真を複数追加
   - 「これらの画像を基に規格の違いを説明して」とプロンプト

3. **音声ファイル**
   - 説明を録音した音声ファイルを追加
   - Video Overviewと同様の機能が利用可能

### 次のステップ

1. 動画を準備する（または既存動画を探す）
2. NotebookLMでノートブックを作成
3. 動画を追加してVideo Overviewを生成
4. 生成された内容を既存資料と統合
5. 最終的な資料をMarkdownやHTMLで整理
