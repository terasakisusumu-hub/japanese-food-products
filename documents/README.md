## 📝 ドキュメント作成ガイドライン

### フォーマット
- 原則としてMarkdown形式（.md）で作成
- 告知やランディングページなど表現重視のコンテンツはHTML形式も許容
- 見出しは階層構造を意識して使用
  - `■` 大見出し
  - `◎` 中見出し
  - `・` 小見出し

### ファイル命名規則
- 日本語ファイル名を使用可能
- 内容が分かりやすい具体的な名前を付ける
- 複数単語は`_`（アンダースコア）で区切る

### 更新履歴
各ドキュメントには作成日または最終更新日を記載すること
